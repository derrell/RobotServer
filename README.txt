- Start Eclipse 3.6

- If the "Welcome to the Eclipse IDE for Java Developers" graphic appears,
  click on the right-most circle, which will say "Workbench" when you mouse
  over it.

- Select File -> Import,

- In "Select an import source, select
  General -> Existing Projects into Workspace

- Click "Next"

- Click on "Select archive file" and then Browse to select your archive
  file.

- Click "Finish"
